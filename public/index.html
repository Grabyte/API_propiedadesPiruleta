Bienvenido a la API de Propiedades Piruleta

Esta API contiene los datos de "propiedades inmobiliarias" para practicar desarrollo web.

Rutas disponibles:

  • /api – Documentación e información más tecnica sobre como utilizar las rutas disponibles.
  • /api/properties – Lista completa de propiedades (JSON).
  • /api/properties/:id – Detalles de una propiedad por ID (por defecto esta ruta va al id 1).
  • /api/properties/summary – Lista resumida (vista tipo tarjeta).
