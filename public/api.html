Documentación de la API

Esta API proporciona acceso a un conjunto de propiedades simuladas en formato JSON.

Rutas disponibles

  • GET /api/properties
    Devuelve un array con todas las propiedades disponibles.
  • GET /api/properties/:id
    Devuelve los detalles completos de una propiedad según su ID.
  • GET /api/properties/summary
    Devuelve un resumen de las propiedades con información básica (ideal para vistas tipo tarjeta).
